## Summary

The complete torque specs for Jeep Wrangler JL vehicles is not freely available
from an official source. Community members have put together this information
in forum posts. The goal of this project is to provide a printable PDF.

If you do not wish to print the data, it can be viewed in your browser at
either of the previously mentioned links, or by viewing the
[torque-specs.md](./torque-specs.md) file in this repository.

You can download a PDF of the torque specs from this repository's
[releases](https://github.com/mssalvatore/jeep-wrangler-jl-torque-specs/releases/latest).

## Build instructions

To build/render the PDF, run the [render-html.sh](./render-html.sh) script to
generate an HTML file at `./artifacts/torque-specs.html`. Point your browser at
this file and use your browser's "print-to-pdf" feature to render the HTML as a
PDF.
